# 📝 Text to PDF Converter

Это веб-приложение на Laravel, позволяющее пользователям вводить текст, сохранять его в базе данных и экспортировать в PDF.

## 🚀 Функционал

- Форма для ввода текста.
- Сохранение текста в базу данных.
- Вывод списка сохранённых данных.
- Генерация PDF и скачивание.
- Использование **Laravel 10+**, **Eloquent ORM**, **Blade**, **Barryvdh/DomPDF**.
